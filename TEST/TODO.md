# Testovanie schopností

1. **Zoznam súborov**: Rekurzívne listujem súbory na pracovnej ploche.
2. **Čítanie súborov**: Čítam UTF-8 textové súbory.
3. **Hľadanie textu**: Hľadám literálne texty v textových súboroch.
4. **Zápis súborov**: Vytvaram alebo nahradzam UTF-8 textové súbory.
5. **Spúšťanie príkazov**: Spúšťam príkazy.
6. **Ukladanie dovedností**: Ukladam opakované miesto ako Markdown pokyny pre budúce úlohy.
